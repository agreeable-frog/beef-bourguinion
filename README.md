# Panoramic view OpenGL

### Libraries used

* GLFW
* GLEW
* GLM
* ImGui

### Features

* Panoramic view with first person view on another window  
* The panoramic view is configurable from the interface (heading, pitch, fov, size)
* The first camera can be moved at the same time
